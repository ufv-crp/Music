## Quick start :computer::wrench:

``` bash
# clone the repository
$ git clone https://github.com/ufv-crp/web-music.git web-music
$ cd web-music

# install dependencies
$ npm i

# dev mode
$ npm start

# build for production
$ npm build

# test script
$ npm test
```
