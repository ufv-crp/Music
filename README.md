### Quick start :wrench:

``` bash
# Clone the repository
$ git clone https://github.com/ufv-crp/web-music.git web-music
$ cd web-music

# Install the dependencies
$ npm i

# Start the serve
$ npm start

# Generate a build
$ npm build

# Run the tests
$ npm test
```
