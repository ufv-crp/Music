### Quick start

```sh
# Clone the repository
$ git clone https://github.com/ufv-crp/web-music.git web-music --depth=1

# Open the repository
$ cd web-music

# Install the dependencies
$ yarn

# Start the serve
$ yarn start

# Generate a build
$ yarn build

# Run the tests
$ yarn test
```
